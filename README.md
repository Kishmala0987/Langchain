# LangChain Learning Repository

This repository contains my hands-on learning and experimentation with **LangChain**.
Each folder in this repository represents a different **LangChain concept, feature, or integration** that I explored while learning the framework.

The goal of this project is to keep practical code examples for important LangChain components in one place so they can be easily revisited, understood, and reused.

---

## 📚 Repository Structure

Each directory focuses on a specific concept or capability within LangChain.

Examples of topics included in this repository may include:

* Prompt Templates
* LLM Chains
* Sequential Chains
* Retrieval Augmented Generation (RAG)
* Vector Databases
* Document Loaders
* Text Splitters
* Embeddings
* LangChain Integrations

Every folder contains:

* Example scripts
* Experimentation code
* Small demonstrations of how the concept works
